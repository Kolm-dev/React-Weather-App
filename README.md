
# React Weather App

Приложения для просмотра текущей температуры в городе

## Команды

Запуск локального сервера для разработки

```bash
  npm install
  npm start
```
    
Продакшн версия проекта

```bash
  npm install
  npm build
```
    

## Технологии

**Front-end:** ReactJS, axios, antd
